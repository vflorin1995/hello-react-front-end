# 📖 Hello React FrontEnd <a name="about-project"></a>

**Hello React FrontEnd** is a react frontend to connect with a rails backend. It fetches a random greeting from the rails api application. 
## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- CSS
- HTML
- React & Redux

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

- A web browser
- A code editor

### Setup

Clone this repository to your desired folder: 

```
git clone https://github.com/vflorin1995/hello-rails-react
```


### Install

Then install the project's necessary dependencies with: 

```
npm install
```

### Run 

Run the app with: 
```
npm run start
```
